## 📖 Описание программы

**Book Tracker** - это графическое приложение на Python для учета прочитанных книг. Программа позволяет:

- ✅ Добавлять книги с указанием названия, автора, жанра и количества страниц
- ✅ Просматривать список всех добавленных книг в удобной таблице
- ✅ Фильтровать книги по жанру и количеству страниц
- ✅ Удалять книги из списка
- ✅ Сохранять данные в JSON-файл и загружать из него
- ✅ Проверять корректность вводимых данных

Приложение имеет интуитивно понятный интерфейс и подходит для личного использования.
